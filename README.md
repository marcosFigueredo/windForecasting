# Time Series Reconstruction and Forecasting with LSTM and Wavelet Decomposition

This repository contains datasets, notebooks, trained models, logs, and visual outputs related to a research study focused on reconstructing and forecasting environmental time series using Long Short-Term Memory (LSTM) networks and Wavelet decomposition techniques.

## 📘 About the Research

This work is the result of the master's research conducted by **Marcos Felipe** in the **Graduate Program in Modeling and Simulation of Biosystems** at the **State University of Bahia (UNEB), Campus Alagoinhas, Bahia, Brazil**.

The goal of the research is to improve time series forecasting by combining wavelet-based decomposition with LSTM models to better capture local temporal features and improve predictive accuracy.

## 🗂 Repository Structure

- `notebook/` – Jupyter notebooks for data preprocessing, modeling, and analysis  
- `models/` – Trained model files  
- `logs/` – Training logs and outputs  
- `requirements.txt` – Python dependencies  
- `.gitignore` – Files/folders excluded from Git  
- `README.md` – Project overview (this file)

## 📊 Contents

- Notebooks: Series reconstruction, wavelet decomposition, outlier removal, LSTM model training.
- Models: Pre-trained LSTM models for specific forecast tasks.
- Figures: Prediction vs. actual graphs, wavelet analysis plots.
- Logs: Training metrics and model diagnostics.

## 🔧 How to Use

